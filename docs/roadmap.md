# Roadmap

Last Updated: 2026-06-14

## Now

- **Chunk Two** — Single app shell: FastAPI backend + React/Vite frontend, five-tab cockpit, single launcher

## Next

- **Chunk Three** — Ask interface: graph-backed Q&A via `graphify query/path/explain`, session transcripts
- **Chunk Four** — Readable map: clustered project-level Cytoscape.js view, click-to-inspect, filters
- **Chunk Five** — Decision ledger: persistent classifications, map badges

## Soon

- **Chunk Six** — Recommendation queue: Ollama-backed cards, accept/reject/defer
- **Chunk Seven** — Steady work mode: bounded background analysis missions

## Later

- **Chunk Eight** — Approved actions: dry-run + approval-gated execution, rollback notes
- **Chunk Nine** — GitHub packaging: public-safe README, demo graph, CI, clean install path

## Non-Goals (MVP)

- Cloud sync or multi-user support
- Editing arbitrary source files from the UI
- Autonomous commits, pushes, or deletes
- Whole-workspace semantic re-extraction from the UI
- Replacing Codex or Claude
